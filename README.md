# DataLib Project



## Возможности
- Проверка корректности даты
- Определение високосного года
- Вычисление дня недели (алгоритм Зеллера)
- Разница между датами
- Добавление и вычитание дней
- Сравнение дат
- Форматированный вывод

## Требования
- Компилятор C++ (GCC / MSVC)
- CMake версии 3.10+
